5.1 Grid Terminology & Lines

The Matrix Protocol: Establishing the two-dimensional coordinate system of modern layout.

The 2D Shift

CSS Grid is the most powerful layout system ever introduced to the browser. While Flexbox manages space along a single axis (row OR column), Grid manages both axes simultaneously.

In the ROSETTA project, Grid is used for the "Grand Layout"—orchestrating the relationship between the sidebar, the header, and the multi-column artifact gallery.

1. The Anatomy of a Grid

To communicate as an architect, you must use precise terminology:

  • Grid Container: The parent element where display: grid is applied.
  • Grid Lines: The horizontal and vertical dividers. They are 1-indexed (1, 2, 3...).
  • Grid Track: The space between two adjacent lines (essentially a "row" or "column").
  • Grid Cell: The smallest unit of a grid (where one row and one column intersect).
  • Grid Area: Any rectangular space composed of one or more grid cells.
CSS matrix_definition.css 
.artifact-matrix {
    display: grid;
    /* Creating 3 columns of equal width */
    grid-template-columns: 1fr 1fr 1fr;
    /* Creating 2 rows of fixed height */
    grid-template-rows: 200px 200px;
    gap: 20px;
}

2. Line-Based Positioning

Unlike Flexbox, where items flow naturally, Grid allows you to "pin" an item to specific line numbers. A grid line exists at the very start, between tracks, and at the very end.

⚡
The Architect's Rule: An item spans from a Start Line to an End Line. If you want an item to cover two columns, it goes from line 1 to line 3.
CSS positioning_logic.css 
.hero-artifact {
    /* Spans from vertical line 1 to 3 */
    grid-column-start: 1;
    grid-column-end: 3;
    
    /* Spans from horizontal line 1 to 2 */
    grid-row-start: 1;
    grid-row-end: 2;
}

/* Shorthand Syntax: start / end */
.hero-artifact-alt {
    grid-column: 1 / 3;
    grid-row: 1 / 2;
}

3. The Negative Index

A unique feature of Grid is the ability to count backward from the end of the grid using negative numbers. -1 always represents the last line of the grid, regardless of how many tracks exist.

CSS reverse_addressing.css 
.full-width-banner {
    /* Spans from the very first line to the very last line */
    grid-column: 1 / -1;
}

4. Spanning logic

If you don't know the line numbers, you can use the span keyword to tell an item how many "cells" to occupy relative to its current position.

CSS spanning_syntax.css 
.wide-card {
    /* Start wherever the flow puts you, but take up 2 columns */
    grid-column: span 2;
}
⚠
Visualization Tip: CSS Grid is invisible by default. To debug your architecture, open your Browser DevTools (F12), go to the "Elements" tab, and click the small "grid" badge next to your container. This will overlay the lines and line numbers directly on your screen.
