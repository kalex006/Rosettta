2.2 Meta Views: Charset & Viewport

The Lens and the Dictionary: Defining global character support and mobile geometry.

The Universal Dictionary: UTF-8

Computers do not understand text; they only understand numbers (bytes). Character Encoding is the map that translates bytes into characters on the screen.

In the early web, we had fragmented standards (ASCII, ISO-8859-1, Windows-1252). Today, we strictly use UTF-8. It is a variable-width encoding capable of representing every character in the Unicode standard—from English letters to Japanese Kanji to Emojis (✨).

The Viewport History

Before the smartphone era, websites were designed for desktop monitors (usually 980px wide). When the iPhone launched, it needed a way to display these large sites on a small screen.

Apple introduced the concept of the Virtual Viewport. Without instruction, a mobile browser pretends to be a desktop monitor (980px wide) and zooms out to fit everything, making text unreadably small.

Controlling the Geometry

To tell the browser "this site is optimized for mobile," we must override the default virtual viewport using the meta viewport tag.

HTML viewport_standard.html 
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Anatomy of the Directives

  • width=device-width: This sets the width of the page to follow the screen-width of the device. It aligns CSS pixels with Device Independent Pixels.
  • initial-scale=1.0: Sets the initial zoom level when the page is first loaded.

The Accessibility "Anti-Pattern"

You may encounter legacy code that disables user zooming to make a web app feel like a native app.

Bad Practice do_not_use.html 
<!-- ❌ NEVER DO THIS -->
<meta name="viewport" content="..., user-scalable=no, maximum-scale=1.0">
⚠
Accessibility Violation: Preventing zoom (`user-scalable=no`) is a violation of WCAG (Web Content Accessibility Guidelines). It effectively blocks visually impaired users from magnifying text to read your content. Modern browsers (like iOS Safari) now ignore this tag entirely to protect users.
