1.1 The History of SGML & HTML

ARCHIVE REF: 1986-1991 | SUBJECT: HYPERTEXT ORIGINS

The SGML Metalanguage

Before HTML existed, the world relied on SGML (Standard Generalized Markup Language). SGML is not a language itself, but a metalanguage—a system used to define other markup languages. It established the concept of the DTD (Document Type Definition), which dictates the rules of a document's structure.

Architect's Insight

HTML was originally just an SGML application. This is why early HTML required a <!DOCTYPE> declaration; it was the pointer telling the browser which SGML rules to follow to parse the document.

The Berners-Lee Synthesis

In 1991, Tim Berners-Lee released "HTML Tags," a simplified subset of SGML designed for research sharing. While SGML was incredibly complex and rigorous, HTML was designed to be resilient—allowing the web to scale even if developers made small syntax errors.

  • 1991: HTML Tags (The birth).
  • 1995: HTML 2.0 (The first formal standard).
  • 1999: HTML 4.01 (The gold standard for a decade).
  • 2014: HTML5 (The shift to a "Living Standard").
