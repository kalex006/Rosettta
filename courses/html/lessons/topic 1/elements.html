1.3 Elements, Tags & Attributes

The Trinity of Syntax: Defining the distinction between the blueprint and the building.

The Ontological Distinction

In the rendering engine, there is a distinct separation between what you type and what exists in memory.

  • Tags: The syntactic markers in the text file (e.g., <p>).
  • Elements: The actual nodes created in the DOM tree (e.g., HTMLParagraphElement).

Think of the Tag as the architectural blueprint, and the Element as the physical room constructed from it.

HTML syntax_anatomy.html 
<!-- This is the Start Tag -->
<p class="intro">

    <!-- This is the Content (Text Node) -->
    The browser reads this text and creates a HTMLParagraphElement.

<!-- This is the End Tag -->
</p>

The Void Elements

Not all elements require a closing sequence. Void elements are elements that, by definition, cannot contain child nodes (neither text nor other elements). Because they have no content, they require no end tag.

Common void elements include <img>, <br>, <input>, and <meta>.

Attribute Architecture

Attributes provide additional information about an element or configure its behavior. They exist in three states:

  • Key-Value Pair: class="container" (The standard configuration).
  • Boolean: disabled or checked. Their presence implies true.
  • Empty: alt="". Used when a value is structurally required but semantically empty.
HTML attribute_syntax.html 
<!-- 1. Standard Quoted (Recommended) -->
<input type="text" name="username">

<!-- 2. Unquoted (Valid in HTML5, but dangerous) -->
<input type=text value=yes>

<!-- 3. Boolean Attribute (No value needed) -->
<input type="checkbox" checked disabled>

Case Sensitivity Rules

The HTML parser is remarkably forgiving, but strictness ensures longevity.

  • Tags: Case-insensitive (`<DIV>` is the same as `<div>`). Standard practice is strictly lowercase.
  • Attributes Names: Case-insensitive (`HREF` vs `href`). Lowercase is standard.
  • Attribute Values: Case-sensitive depending on context (e.g., IDs and Classes are strictly case-sensitive).
