6.4 *args and **kwargs

Sometimes you do not know how many arguments will be passed into your function. Python provides two special symbols for this.

Arbitrary Arguments (*args)

If you add * before the parameter name, the function will receive a tuple of arguments.

def sum_all(*nums):
    return sum(nums)

print(sum_all(1, 2, 3, 4, 5)) # Output: 15

Keyword Arguments (**kwargs)

If you add ** before the parameter name, the function will receive a dictionary of arguments.

def user_profile(**data):
    print(data["name"])

user_profile(name="Kalex", age=20)
