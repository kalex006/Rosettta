6.5 Lambda Functions

A Lambda function is a small anonymous function. It can take any number of arguments, but can only have one expression.

Syntax

# lambda arguments : expression

Example

# A standard function
def square(n):
    return n * n

# The same as a Lambda
square_lambda = lambda n : n * n

print(square_lambda(5)) # Output: 25

Lambda functions are often used when you need a simple function for a short period of time, especially inside functions like filter() or map().
