6.3 Return Values

To let a function return a value back to the caller, use the return statement.

The Return Statement

Unlike print(), which just shows text on the screen, return sends data back so it can be stored in a variable.

def add_numbers(x, y):
  return x + y

result = add_numbers(5, 3)
print(result) # Output: 8
Crucial Note: Once a function hits a return statement, it exits immediately. Any code written after the return inside that function will never run.
