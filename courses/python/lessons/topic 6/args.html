6.2 Arguments & Parameters

Information can be passed into functions as arguments. Arguments are specified after the function name, inside the parentheses.

Parameters vs Arguments

A Parameter is the variable listed inside the parentheses in the function definition. An Argument is the value that is sent to the function when it is called.

def greet(fname): # fname is the parameter
  print("Hello " + fname)

greet("Kalex")   # "Kalex" is the argument
greet("Rosetta")

Default Parameter Value

You can assign a default value to a parameter. If the function is called without an argument, it uses the default.

def my_country(country = "India"):
  print("I am from " + country)

my_country("Sweden")
my_country() # Output: I am from India
