2.2 If, Elif, Else Statements

Conditionals allow your program to take different paths depending on whether a condition is True or False.

The Structure

if condition:
    # Execute this block if condition is True
elif another_condition:
    # Execute this block if the first condition was False
else:
    # Execute this block if nothing above was True

Example:

age = 18

if age >= 18:
    print("You can vote!")
elif age == 17:
    print("Almost there.")
else:
    print("Sorry, too young.")
Python Note: Indentation (whitespace) is crucial. Without correct indentation, Python will raise an IndentationError.
