2.4 Match Case (Switch)

Introduced in Python 3.10, match is similar to the switch statement found in other languages like C or Java. It is cleaner than writing many if-elif blocks.

Basic Syntax

status = 404

match status:
    case 400:
        print("Bad Request")
    case 404:
        print("Not Found")
    case 418:
        print("I'm a teapot")
    case _:
        print("Something's wrong with the internet")

The _ acts as a wildcard (like else or default), catching anything not matched above.
