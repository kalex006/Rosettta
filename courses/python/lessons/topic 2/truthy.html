2.5 Truthy & Falsy Values

In Python, every value evaluates to either True or False when used in a boolean context (like an if statement).

Falsy Values

These values are considered False:

  • False
  • None
  • 0 (Zero of any numeric type)
  • "" (Empty string)
  • [], (), {} (Empty sequences/collections)

Truthy Values

Almost everything else is considered True.

name = ""
if name:
    print("Name is present")
else:
    print("Name is missing") 
    # Output: Name is missing because empty string is Falsy
