2.1 Booleans & Comparison

Booleans represent one of two values: True or False. They are fundamental for making decisions in your code.

Comparison Operators

These operators compare two values and return a Boolean result:

  • == : Equal to (e.g., 5 == 5 is True)
  • != : Not equal to (e.g., 5 != 3 is True)
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to
print(10 > 9)   # True
print(10 == 9)  # False
print(10 < 9)   # False
