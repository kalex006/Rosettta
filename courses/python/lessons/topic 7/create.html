7.3 Creating Your Own Modules

You can turn any Python file into a module simply by saving it with a .py extension and importing it in another file.

Step 1: Create `tools.py`

# tools.py
def say_hi(name):
    print(f"Hello, {name}!")

version = "1.0.0"

Step 2: Use it in `main.py`

# main.py
import tools

tools.say_hi("Kalex")
print(tools.version)
Pro Tip: When you import a module, Python creates a folder named __pycache__. Don't worry about it; it contains compiled code to make your imports faster next time.
