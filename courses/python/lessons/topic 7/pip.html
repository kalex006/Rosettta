7.4 Pip & PyPI (The World of Packages)

PyPI (Python Package Index) is a repository of software for the Python programming language. pip is the package manager used to install them.

Installing a Library

To install a library like requests (for web API calls), you run this in your terminal:

pip install requests

Using an Installed Library

import requests

response = requests.get("https://api.github.com")
print(response.status_code) # 200

Useful Commands

  • pip list: See all installed packages.
  • pip uninstall name: Remove a package.
  • pip install -U name: Upgrade a package.
