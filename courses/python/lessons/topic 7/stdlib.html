7.2 Standard Library Essentials

Python comes with a "Batteries Included" philosophy. This means it has a massive Standard Library already installed with the language.

The math Module

import math
print(math.ceil(4.2))  # 5 (Rounds up)
print(math.floor(4.8)) # 4 (Rounds down)
print(math.pow(2, 3))  # 8.0

The random Module

Used for generating pseudo-random numbers.

import random

print(random.random())      # Random float between 0.0 and 1.0
print(random.randint(1, 10)) # Random integer between 1 and 10
items = ["A", "B", "C"]
print(random.choice(items)) # Picks a random item
