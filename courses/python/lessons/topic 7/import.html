7.1 The Import System

In Python, a Module is simply a file containing Python code. The import statement allows you to use code from one module in another.

Basic Import

import math
print(math.sqrt(16)) # Output: 4.0

Aliasing (as)

You can create an alias when you import a module by using the as keyword. This is common for long module names.

import math as m
print(m.pi)

Importing Specific Parts

If you only need one function from a big module, use from.

from math import pi, sqrt
print(pi) # No need to write math.pi anymore
Warning: Avoid from module import *. It imports everything and can cause "Naming Collisions" (where two different modules have a function with the same name).
