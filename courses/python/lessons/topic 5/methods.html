5.2 Dictionary Methods

Python provides several methods to interact with dictionaries safely and efficiently.

Safe Access with .get()

Using dict["key"] will crash if the key doesn't exist. Using .get() returns None instead.

print(user.get("phone")) # None (No error!)
print(user.get("phone", "Not Provided")) # Default value

Extracting Data

print(user.keys())   # Get all keys
print(user.values()) # Get all values
print(user.items())  # Get all key-value pairs as tuples

Removing Items

user.pop("year") # Removes specific key
del user["name"] # Another way to delete
user.clear()     # Empties the dictionary
