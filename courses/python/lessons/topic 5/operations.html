5.4 Set Operations

Python sets support mathematical operations like Union and Intersection, which are incredibly fast and useful for comparing data.

Common Operations

set_a = {1, 2, 3}
set_b = {3, 4, 5}

# Union (Combine both)
print(set_a | set_b) # {1, 2, 3, 4, 5}

# Intersection (Items in both)
print(set_a & set_b) # {3}

# Difference (Items in A but NOT in B)
print(set_a - set_b) # {1, 2}
