5.1 Dictionaries (Key-Value)

A Dictionary is a collection which is ordered, changeable, and does not allow duplicates. They store data in key:value pairs.

Basic Syntax

Dictionaries are written with curly brackets {}.

user = {
  "name": "Kalex",
  "year": 2026,
  "is_admin": True
}

print(user["name"]) # Output: Kalex

Accessing & Changing

You can access items by referring to their key name inside square brackets.

user["year"] = 2027 # Changing a value
user["email"] = "kalex@rosetta.com" # Adding a new key
Note: As of Python 3.7, dictionaries are ordered. Before that, they were unordered.
