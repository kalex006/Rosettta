5.3 Sets (Unique Items)

A Set is a collection which is unordered, unindexed, and does not allow duplicate values.

Syntax

Sets are also written with curly brackets {}, but they only have values, no keys.

ids = {101, 102, 103, 101}
print(ids) # Output: {101, 102, 103} (Duplicate 101 is gone!)

Characteristics

  • Unordered: You cannot be sure in which order the items will appear.
  • Unchangeable: You cannot change items, but you can add/remove them.
  • No Duplicates: Perfect for filtering out repetitive data.
ids.add(104)    # Add one item
ids.remove(101) # Remove an item
