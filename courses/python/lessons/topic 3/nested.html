3.4 Nested Loops

A nested loop is a loop inside another loop. The "inner loop" will be executed one time for each iteration of the "outer loop".

Example: Coordinates

for x in range(3):       # Outer loop
    for y in range(2):   # Inner loop
        print(f"({x}, {y})")

Output:

(0, 0)
(0, 1)
(1, 0)
(1, 1)
(2, 0)
(2, 1)

Nested loops are commonly used for working with 2D data like grids, tables, or images.
