3.5 Loop Else Clause

Python has a unique feature: you can use an else block with a loop. The else block runs only if the loop finished naturally (not by a break).

Example: Searching

for n in range(5):
    print(n)
else:
    print("Loop finished successfully!")

When it does NOT run:

for n in range(5):
    if n == 3:
        break
    print(n)
else:
    print("This will NOT print because of the break.")
Pro Tip: Use the loop-else clause when searching for something. If the item isn't found (no break), you can use the else block to show a "Not Found" message.
