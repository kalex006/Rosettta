3.2 For Loops & Range

A for loop is used for iterating over a sequence (like a list, a tuple, a dictionary, a set, or a string).

The Range Function

To loop through a set of code a specified number of times, we use the range() function.

# Loops from 0 to 4 (5 is not included)
for x in range(5):
    print(x)

# range(start, stop, step)
for y in range(2, 10, 2):
    print(y) # Output: 2, 4, 6, 8

Iterating over Strings

for letter in "ROSETTA":
    print(letter)
