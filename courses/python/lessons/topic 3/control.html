3.3 Loop Control: Break & Continue

Sometimes you need to stop a loop early or skip certain parts of it. We use break and continue for this.

The Break Statement

Used to exit the loop immediately, even if the condition is still True.

for x in range(10):
    if x == 5:
        break # Stops the loop when x reaches 5
    print(x)

The Continue Statement

Used to skip the rest of the current iteration and move to the next one.

for x in range(5):
    if x == 3:
        continue # Skips printing 3
    print(x)
