3.1 While Loops

A while loop repeats a block of code as long as a specified condition is True.

Basic Syntax

i = 1
while i < 6:
    print(i)
    i += 1  # Increment i, otherwise the loop runs forever!

The Infinite Loop

If the condition never becomes False, you get an Infinite Loop. This can crash your program. Always ensure your loop has a way to exit.

Note: In Python, there is no i++ operator. You must use i += 1 or i = i + 1.
