8.2 Writing & Appending Files

To write to an existing file, you must add a parameter to the open() function: "a" or "w".

The Difference

  • "a" - Append: Adds text to the end of the file.
  • "w" - Write: Overwrites the entire file. Be careful!
# Appending
f = open("log.txt", "a")
f.write("New entry added!\n")
f.close()

# Overwriting
f = open("log.txt", "w")
f.write("Everything else is gone now.")
f.close()
Note: If the file does not exist, both "a" and "w" will create a new empty file for you.
