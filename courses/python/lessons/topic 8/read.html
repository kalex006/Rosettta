8.1 Reading Files

Python has a built-in open() function to open a file. It takes two parameters: **Filename** and **Mode**.

The "r" Mode (Read)

By default, open() uses the "r" mode, which stands for Read.

# Opening a file named "data.txt"
f = open("data.txt", "r")

# Read the entire content
print(f.read())

# Read only the first 5 characters
# print(f.read(5))

# Always close the file when done!
f.close()

Reading Line by Line

f = open("data.txt", "r")
for line in f:
    print(line)
f.close()
