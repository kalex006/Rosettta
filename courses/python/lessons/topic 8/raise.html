8.5 Raising & Finally

Sometimes you want to create your own error if a specific condition is met. We use the raise keyword for this.

Raising an Error

age = -5
if age < 0:
    raise Exception("Age cannot be negative!")

The Finally Block

The finally block, if specified, will be executed regardless if the try block raises an error or not.

try:
    f = open("data.txt")
    # do something
except:
    print("Error!")
finally:
    print("Closing resources...")
    # This always runs
Pro Tip: Use finally for "Cleanup" tasks, like closing a database connection or a network socket.
