1.2 Variables & Assignment

Variables are containers for storing data values. Unlike other languages (like C++ or Java), Python has no command for declaring a variable.

Creating Variables

A variable is created the moment you first assign a value to it using the = operator.

x = 5
name = "Kalex"
is_active = True

print(x)
print(name)

Dynamic Typing

Python is dynamically typed. This means you don't need to state the type (int, string, etc.), and you can even change the type later.

x = 4       # x is of type int
x = "Sally" # x is now of type str
print(x)

Naming Rules

  • Must start with a letter or underscore (_).
  • Cannot start with a number.
  • Case-sensitive (age, Age, and AGE are three different variables).
  • Convention: Use snake_case for variables (e.g., my_variable_name).
