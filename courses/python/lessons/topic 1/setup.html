1.1 Setup & Hello World

Python is an interpreted, high-level programming language known for its simplicity and readability. Before we code, we must ensure Python is installed.

Checking Installation

Open your terminal or command prompt and type:

python --version

If you see a version number (like Python 3.12.0), you are ready. If not, download it from python.org.

Your First Script

In Python, we use the print() function to output text to the console.

# This is a comment. Python ignores lines starting with #.
print("Hello, World!")

Running the Code

Save your file as main.py and run it in the terminal:

python main.py

Output:

Hello, World!
Fun Fact: Python is not named after the snake, but after the British comedy group "Monty Python's Flying Circus."
