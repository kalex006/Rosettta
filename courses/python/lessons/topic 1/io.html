1.4 Input & Output

Software is useless if it cannot interact with the user. We need ways to get data IN and send data OUT.

Output: The Print Function

We already know print(). But it can do more.

print("Hello", "World", sep="-") 
# Output: Hello-World

Input: The Input Function

The input() function pauses your program and waits for the user to type something.

name = input("Enter your name: ")
print("Hello, " + name)
Crucial Note: The input() function ALWAYS returns a string. If you want a number, you must convert it:
age = int(input("Enter age: "))
