4.1 Creating Lists

A List is an ordered, changeable collection of items. It is one of the most used data types in Python.

Basic Syntax

Lists are written with square brackets []. Items are separated by commas.

fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = ["Kalex", 2026, True] # Lists can hold different data types

List Length

To determine how many items a list has, use the len() function:

print(len(fruits)) # Output: 3
Note: Lists are Mutable, meaning you can change, add, or remove items after the list has been created.
