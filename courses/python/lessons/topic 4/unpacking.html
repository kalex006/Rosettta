4.5 Unpacking Sequences

Unpacking allows you to extract values from a list or tuple and assign them to variables in a single line.

Basic Unpacking

point = (4, 5)
x, y = point

print(x) # 4
print(y) # 5

Extended Unpacking (The * Asterisk)

If you have more values than variables, you can use * to collect the remaining items into a list.

fruits = ["apple", "banana", "cherry", "strawberry", "raspberry"]

green, yellow, *red = fruits

print(green)  # "apple"
print(yellow) # "banana"
print(red)    # ["cherry", "strawberry", "raspberry"]
