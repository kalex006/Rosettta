4.3 List Methods

Python has a set of built-in methods that you can use on lists to modify them dynamically.

Adding Items

names = ["Alice", "Bob"]

names.append("Charlie") # Adds to the end
names.insert(1, "Dave") # Inserts "Dave" at index 1
print(names) # ["Alice", "Dave", "Bob", "Charlie"]

Removing Items

names.remove("Bob") # Removes specific item
names.pop()         # Removes the last item
names.pop(0)        # Removes item at index 0
names.clear()       # Empties the list

Sorting

vals = [5, 2, 9, 1]
vals.sort()    # [1, 2, 5, 9]
vals.reverse() # [9, 5, 2, 1]
