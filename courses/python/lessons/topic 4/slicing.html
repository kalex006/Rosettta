4.2 Indexing & Slicing

Python allows you to access specific items or "slices" of a list using their index position.

Positive & Negative Indexing

Indexing starts at 0. Negative indexing starts from the end (-1).

colors = ["red", "green", "blue", "yellow"]

print(colors[0])  # "red"
print(colors[-1]) # "yellow" (the last item)

Slicing Syntax

You can specify a range of indexes by specifying where to start and where to end the range: [start:stop].

nums = [10, 20, 30, 40, 50]

print(nums[1:4]) # [20, 30, 40] (index 4 is excluded)
print(nums[:3])  # [10, 20, 30] (start to index 3)
print(nums[2:])  # [30, 40, 50] (index 2 to end)
