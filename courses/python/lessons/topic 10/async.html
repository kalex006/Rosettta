10.4 Introduction to AsyncIO

AsyncIO is a library to write concurrent code using the async/await syntax. It allows your program to handle multiple tasks at once without waiting.

Async & Await

import asyncio

async def fetch_data():
    print("Start fetching...")
    await asyncio.sleep(2) # Simulates a network delay
    print("Data fetched!")

asyncio.run(fetch_data())

When to use it?

Use AsyncIO for I/O-bound tasks (tasks that wait for something else):

  • Making Web API calls.
  • Reading large files from a disk.
  • Database queries.
