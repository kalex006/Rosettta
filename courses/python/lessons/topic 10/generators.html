10.2 Generators (yield)

Generators allow you to declare a function that behaves like an iterator. They are incredibly Memory Efficient.

The "yield" Keyword

Unlike return, which sends back a value and stops, yield pauses the function, saves its state, and can be resumed later.

def count_to_three():
    yield 1
    yield 2
    yield 3

gen = count_to_three()
print(next(gen)) # 1
print(next(gen)) # 2

List vs Generator

A List stores all items in memory (RAM). A Generator produces items one by one on demand. Use Generators when working with massive datasets (millions of lines) to avoid crashing your computer.
