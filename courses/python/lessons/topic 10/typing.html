10.3 Type Hinting

Python is dynamically typed, but modern Python uses Type Hints to make code more readable and to help catch bugs before they happen.

Adding Types

# Variable: type = value
age: int = 25
name: str = "Kalex"

# Function(param: type) -> return_type
def greeting(name: str) -> str:
    return f"Hello {name}"

Benefits

  • Documentation: You immediately know what kind of data a function expects.
  • IDE Support: VS Code can provide better autocomplete suggestions.
  • Static Analysis: Tools like mypy can check your code for type errors.
