10.5 The Zen of Python

Congratulations! You have decoded the Python language. To finish, you must understand the philosophy of Python.

Import This

Open a Python terminal and type import this. You will see a poem called The Zen of Python by Tim Peters.

Core Principles

  • Beautiful is better than ugly.
  • Explicit is better than implicit.
  • Simple is better than complex.
  • Readability counts.
  • If the implementation is hard to explain, it's a bad idea.
