9.5 Polymorphism

The word "Polymorphism" means "many forms". In programming, it refers to methods or functions that can do different things based on the object they are acting upon.

Same Method, Different Class

Imagine different animals. They all have a move() method, but they move differently.

class Car:
  def move(self): print("Drive!")

class Boat:
  def move(self): print("Sail!")

class Plane:
  def move(self): print("Fly!")

# Iterate through different objects
for x in (Car(), Boat(), Plane()):
  x.move() # Same call, different results
Advanced: Polymorphism allows you to write code that can handle many types of objects without knowing their exact class beforehand.
