9.4 Inheritance

Inheritance allows us to define a class that inherits all the methods and properties from another class.

Parent vs Child Class

  • Parent Class: The class being inherited from (Base Class).
  • Child Class: The class that inherits from another class (Derived Class).
class Animal: # Parent
    def speak(self):
        print("Animal makes a sound")

class Cat(Animal): # Child inherits from Animal
    def meow(self):
        print("Meow!")

my_cat = Cat()
my_cat.speak() # Inherited method
my_cat.meow()  # Own method
