9.3 Methods (Object Behavior)

Objects can also contain methods. Methods in objects are functions that belong to the object.

Defining Methods

Methods are defined inside the class and always take self as the first argument.

class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        print(f"{self.name} says Woof!")

my_dog = Dog("Buddy")
my_dog.bark() # Buddy says Woof!

Modifying Object Properties

You can change properties on objects like this:

my_dog.name = "Max"
# Now my_dog.bark() will say "Max says Woof!"
