9.1 Classes & Objects

Python is an object-oriented programming language. Almost everything in Python is an Object, with its properties and methods.

What is a Class?

A Class is like a blueprint or a constructor for creating objects. Think of a "Car" blueprint. From one blueprint, you can build many "Car" objects (Red Car, Blue Car, Fast Car).

Basic Syntax

Classes are created using the class keyword. Conventionally, Class names start with a Capital Letter.

class MyClass:
  x = 5

# Creating an Object (Instance)
p1 = MyClass()
print(p1.x) # Output: 5
Real World Analogy: A Class is the drawing of a house. An Object is the actual house built from that drawing.
