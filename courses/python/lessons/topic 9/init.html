9.2 The __init__ Method (Constructors)

All classes have a function called __init__(), which is always executed when the class is being initiated.

Assigning Values

Use the __init__() function to assign values to object properties, or other operations that are necessary to do when the object is being created.

class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

p1 = Person("Kalex", 2026)

print(p1.name)
print(p1.age)

What is "self"?

The self parameter is a reference to the current instance of the class. It is used to access variables that belong to the class. It does not have to be named self, but it is the universal standard in Python.
